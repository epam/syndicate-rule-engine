# link_tenant

## Description

Links Maestro tenant to CUSTODIAN parent

## Synopsis

```bash
c7n parent link_tenant
    --parent_id <text>
    [--tenant_name <text>]
```

## Options

`--parent_id` (text) 

Maestro Parent id to link to tenant.

`--tenant_name` (text) 

Name of related tenant
